### 步骤
* 全局安装 docsify-cli 工具

```
npm i docsify-cli -g
```
* 初始化项目

```
docsify init
```
* 指定端口启动项目，默认3000

```
docsify serve -p 3000
```
* 本地预览

```
http://localhost:3000
```
